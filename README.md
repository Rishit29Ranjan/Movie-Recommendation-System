A machine learning-based Movie Recommendation System that suggests movies to users based on their preferences. This project uses content-based filtering techniques to recommend similar movies by analyzing features like genre, keywords, cast, and overview.

The trained similarity model (`similarity.pkl`) is not included in this repository due to GitHub file size limits.

You can download it from the link below:

👉 https://drive.google.com/your-link

After downloading, place the file in the project root directory.
